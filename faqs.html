FAQs

What is DarkroomX?

DarkroomX is an AI-powered photo editing platform designed for speed and creative control. You can upload an image, apply prompt-based edits, generate new visual variations, and export final results without a complex catalog workflow.

How is DarkroomX different from traditional editors?

DarkroomX uses a session-based workflow instead of a persistent catalog model. That means each job stays focused and self-contained: open, edit, export, and move on.

Do I need to install anything?

No. DarkroomX runs in the browser, so you can start from the web app and work directly online.

How do credits work?

DarkroomX uses a credit-based system for actions like AI generation and export. Credit pack options are available on the Pricing page.

Do credits expire?

No. Credits are pay-once and do not expire.

What file types can I work with?

DarkroomX supports common image workflows for upload, editing, and export, including supported RAW formats. If you run into a file-specific issue, contact support and we’ll help verify compatibility.

Does DarkroomX support RAW files?

Yes. You can upload supported RAW formats including DNG, CR2, CR3, NEF, ARW, RW2, ORF, RAF, PEF, SRW, and X3F.

How does RAW support work?

When you upload a RAW file, DarkroomX prepares a high-quality preview for editing in the studio while keeping your session workflow fast and responsive.

Can I edit multiple images at once?

Yes. Batch-oriented workflows are supported for repeated operations across multiple images.

Can I still make manual edits, or is it only AI?

You can do both. DarkroomX is built to balance manual control with AI-assisted speed so you can direct the final look instead of relying on one-click automation.

Does DarkroomX preserve the original image?

We aim to preserve original image structure when requested, but AI-generated edits can introduce visual variation. For legal and technical details, see our Terms of Use.

Can I use DarkroomX for client/commercial work?

Yes. Many workflows are suitable for professional and commercial delivery. You should still review final outputs for brand/legal compliance before publishing.

Is there framed print ordering?

Framed print ordering is planned and currently marked as coming soon.
