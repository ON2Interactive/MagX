Getting Started

New to DarkroomX? Start by creating your account on Sign Up, then open the editor and begin a new session.

  • Upload your image(s)
  • Apply manual or AI-powered edits
  • Generate variations if needed
  • Export your final result

Core Workflow

DarkroomX is built around session-based editing. Each session is a focused workspace for one job, shoot, or concept, without the overhead of traditional catalog management.

Using AI Edits & Generations

In the editor, you can use AI in two ways: edit an existing photo or generate a new image from text. The more specific your prompt, the more predictable your result.

  • Use AI Edit to modify an uploaded image while keeping control over the original composition
  • Use Generate to create new images from text prompts
  • State the subject and desired change directly
  • Include style, tone, or lighting intent when relevant
  • Generate multiple attempts for best selection

RAW Support

DarkroomX supports uploading professional RAW formats and prepares a high-quality editable preview in your session. Once the preview is ready, you can continue editing, generating, and exporting as usual.

  • Supported formats include DNG, CR2, CR3, NEF, ARW, RW2, ORF, RAF, PEF, SRW, and X3F
  • RAW uploads may take a few extra moments while the preview is generated
  • After preview generation, the image behaves like any other studio image

Projects (Library)

DarkroomX uses session-based projects so each shoot or job stays self-contained. Open the Library from the left toolbar to manage your projects.

  • Enter a project name in the Library input
  • Click the Save icon to save your current studio session to that project
  • Click the + icon to create a new project and start fresh
  • Click any project card thumbnail to reopen that saved session

If you refresh the app, reopen your project from Library to continue where you left off.

Wallart (Posters, Canvas, Framed Prints)

DarkroomX lets you order physical prints directly from the studio using the current selected image. You can choose Posters, Canvas, or Framed Prints and then finish product options (size, frame, edge, quantity) on checkout.

  • Select the image you want to print in Studio
  • Click Order Wallart in the top bar (or the print tool)
  • Confirm the preview thumbnail and choose product type
  • Continue to checkout and complete size/frame selections

Tip: the preview shown in the Wallart modal is the exact image being sent for print.

Credits & Billing Help

DarkroomX uses a subscription + credits model. An active subscription unlocks full access to the editor. Credits are used for AI actions, including AI Edits and Image Generations.

If you need more credits, visit Pricing. For subscription, billing, or purchase issues, please reach out through Contact.

Export Help

If an export fails, try reducing file complexity, refreshing the session, and exporting again. If the issue persists, contact support with a short description of what happened.

Performance Tips

  • Use a modern browser and keep it updated
  • Close unused tabs during heavy editing sessions
  • Use stable internet for AI generation tasks

Account & Access
